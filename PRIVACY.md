# Política de Privacidade — Classificador CNAE

**Última atualização:** maio de 2025

## 1. Coleta de Dados

O Classificador CNAE **não coleta, armazena, transmite nem compartilha** nenhum dado pessoal dos usuários.

## 2. Uso do CNPJ

Quando o usuário digita um CNPJ na aba de busca, o número é enviado diretamente às APIs públicas abaixo para consulta de dados cadastrais da Receita Federal:

- [BrasilAPI](https://brasilapi.com.br) — API pública e gratuita
- [ReceitaWS](https://receitaws.com.br) — API pública e gratuita (fallback)

O CNPJ **não é armazenado** localmente nem em nenhum servidor próprio. A extensão não possui servidor backend.

## 3. Dados de Navegação

A extensão **não acessa, monitora nem registra** nenhuma informação sobre a navegação do usuário, histórico de páginas, abas abertas ou qualquer outra atividade no navegador.

## 4. Armazenamento Local

A extensão **não utiliza** cookies, `localStorage`, `sessionStorage` ou qualquer outro mecanismo de armazenamento persistente.

## 5. Terceiros

A extensão se comunica exclusivamente com as APIs públicas mencionadas na seção 2, apenas quando o usuário solicita uma busca por CNPJ. Nenhum dado é enviado a terceiros em nenhuma outra circunstância.

## 6. Crianças

Esta extensão não é direcionada a menores de 13 anos e não coleta dados de nenhum usuário, independentemente da idade.

## 7. Alterações nesta Política

Qualquer alteração nesta política será publicada nesta página com a data de atualização revisada.

## 8. Contato

Em caso de dúvidas sobre esta política de privacidade, abra uma [issue no repositório](../../issues).
